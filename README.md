# rock

# Usage
| short | long        | description                                 | example                           |
|-------|-------------|---------------------------------------------|-----------------------------------|
| `-r`  | `--replace` | replace a certain string with another       | `rock --replace "~" "/home/jake"` |
| `-s`  | `--split`   | split into multiple lines after a character | `rock --split ,`                  |
